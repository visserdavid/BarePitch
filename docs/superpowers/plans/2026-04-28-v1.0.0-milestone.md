## Task 4: Register error pages in .htaccess

**Files:**
- Modify: `public/.htaccess`

- [ ] **Step 1: Add ErrorDocument directives**

Append to the bottom of `public/.htaccess`:

```apache
ErrorDocument 404 /404.html
ErrorDocument 500 /500.html
```

- [ ] **Step 2: Verify**

The PHP built-in server (`php -S`) ignores `.htaccess` — these directives only activate under Apache. Verify the lines are present and correctly spelled.

- [ ] **Step 3: Commit**

```bash
git add public/.htaccess
git commit -m "feat: register 404 and 500 error pages in .htaccess"
```
