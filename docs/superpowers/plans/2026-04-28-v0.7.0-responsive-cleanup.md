# v0.7.0 Responsive Cleanup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make every BarePitch page usable on a 390px mobile screen by adding a single responsive CSS file, a fixed bottom navigation bar, priority-column tables, and tappable rows.

**Architecture:** One new file (`responsive.css`) holds all media-query overrides — existing CSS files are untouched. The bottom nav lives in `footer.php` so it appears everywhere. Table responsiveness is pure CSS (`.col-hide-mobile`) plus a small JS tap-handler written inline in `footer.php`.

**Tech Stack:** Plain PHP 8.2, HTML, CSS (no preprocessor), vanilla JS, no build tools.

## Task 2: Create responsive.css

**Files:**
- Create: `public/assets/css/responsive.css`

- [ ] **Step 1: Create the file with all responsive rules**

Create `public/assets/css/responsive.css` with this exact content:

```css
/* Page header — replaces inline flex styles on list views */
.page-header {
    display: flex;
    align-items: center;
    justify-content: space-between;
    margin-bottom: 1.25rem;
}

/* Bottom nav — hidden on desktop */
.bottom-nav {
    display: none;
}

/* ===== Mobile (≤ 640px) ===== */
@media (max-width: 640px) {

    /* Space for fixed bottom nav */
    body {
        padding-bottom: 56px;
    }

    /* Bottom navigation bar */
    .bottom-nav {
        display: flex;
        position: fixed;
        bottom: 0;
        left: 0;
        right: 0;
        height: 56px;
        background: #f4f4f4;
        border-top: 1px solid #ddd;
        z-index: 100;
    }

    .bottom-nav__item {
        flex: 1;
        display: flex;
        align-items: center;
        justify-content: center;
        text-decoration: none;
        color: #005fcc;
        font-size: 0.875rem;
        font-weight: 600;
    }

    .bottom-nav__item:active {
        background: #e8e8e8;
        color: #003f8a;
    }

    /* Hide lower-priority table columns */
    .col-hide-mobile {
        display: none;
    }

    /* Tappable table rows */
    .table tr[data-href] {
        cursor: pointer;
    }

    .table tr[data-href]:active {
        background: #f0f0f0;
    }

    /* Suppress link affordance inside tappable rows */
    .table tr[data-href] td a {
        pointer-events: none;
        color: inherit;
        text-decoration: none;
    }

    /* Page header stacks vertically */
    .page-header {
        flex-direction: column;
        align-items: flex-start;
        gap: 0.75rem;
    }

    /* Form action buttons wrap */
    .form-actions {
        flex-wrap: wrap;
    }
}
```
